# UI_Start 场景重构计划

## 当前问题分析

1. **按钮样式问题**：当前所有按钮都使用了 StyleBoxEmpty，导致按钮没有背景色和边框，视觉上只是文字浮在背景上
2. **交互反馈不足**：按钮缺少悬停、按下等状态的视觉反馈
3. **古剑logo选择按钮**：仅通过颜色变化表示选中状态，视觉效果不明显
4. **整体UI风格不统一**：各种控件的样式不一致

## 重构目标

1. 为所有交互元素提供清晰的视觉反馈
2. 统一UI风格，使其更加美观
3. 提升用户体验，让界面更具现代感
4. 保持与游戏整体美术风格的一致性

## 具体实施方案

### 1. 开始按钮 (StartButton)
- 使用 StyleBoxFlat 作为背景
- 设置正常状态：深蓝色背景 (#2C3E50)，白色文字
- 设置悬停状态：亮蓝色背景 (#34495E)，白色文字
- 设置按下状态：更深的蓝色背景 (#1A252F)，白色文字
- 添加字体描边效果

### 2. 古剑logo选择按钮 (Gujian1, Gujian2, Gujian3)
- 保持TextureButton的基本功能
- 为选中/未选中状态提供更明显的视觉差异
- 添加选中时的高亮边框效果
- 调整透明度变化的平滑度

### 3. 设置按钮 (SettingButton)
- 使用与开始按钮类似的样式方案
- 保持图标显示
- 添加合适的背景和边框

### 4. 特殊牌复选框 (SpecialCardCheckbox)
- 使用项目中已有的复选框纹理资源
- 改进标签文字的显示效果

### 5. 样式资源定义

#### StyleBoxFlat 定义
- 正常状态：深色半透明背景，圆角矩形
- 悬停状态：稍亮的背景色
- 按下状态：更深的背景色
- 边框：细线边框，增加立体感

## 技术实现细节

### 颜色方案
- 主色调：深蓝灰色系 (#2C3E50, #34495E, #1A252F)
- 强调色：金色或橙色用于选中状态 (#F39C12)
- 文字色：白色或浅灰色

### 字体设置
- 保持现有的思源宋体
- 增加字体描边效果以提高可读性
- 调整字体大小以适应新样式

## 预期效果

重构后，UI_Start场景将具有：
1. 更清晰的视觉层次
2. 更好的交互反馈
3. 更统一的UI风格
4. 更高的美观度
5. 更佳的用户体验
