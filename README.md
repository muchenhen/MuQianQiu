### 项目介绍：

本项目是对《古剑奇谭三：梦付千秋星垂野》中的内置游戏《千秋戏》的复刻，使用UE4引擎的UMG配合sluaUnreal插件进行制作。

主要目的是熟悉UE4的UMG相关知识，不接受任何玩法相关的建议和意见，想改玩法规则的话自己fork。

所有相关素材均为原作公司所有。

#### 项目情况

引擎版本：UE4 4.26
