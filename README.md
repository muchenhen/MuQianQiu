### 项目介绍：

本项目是对《古剑奇谭三：梦付千秋星垂野》中的内置游戏《千秋戏》的玩法复刻。  
UI系统使用UMG搭建  
Lua插件使用[UnLua](https://github.com/Tencent/UnLua)

本项目没有什么目的，就是想做  
玩法采用单机内的版本    
不接受任何玩法相关的建议和意见，想改玩法规则的话自己fork。

**所有相关素材均为原作公司上海烛龙信息科技有限公司所有。**
